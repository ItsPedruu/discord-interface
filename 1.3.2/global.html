    PaginationOptions

    Options for the Pagination Interface
    Properties:
    Name Type Attributes Description
    time number The amount of time in milliseconds until the collectors stops
    removeReactions boolean Whether to remove the reactions from the user when he reacts
    reactions PaginationReactionOptions The pagination reactions for the pagination controllers
    getPage function The function to return the page formattion for the pagination pages as it's changes
    items array The items to be displayed in the pagination
    userID string The id of the discord user that is allowed to control the pagination
    extraReactions array An array of emoji id's or unicodes to be added as extraReactions in the pagination. When the user reacts to the extra reaction it fires the Pagination#collect
